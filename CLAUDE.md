# Project Instructions

## Python Environment

Always use the `pythonenv-lerobot` virtual environment when running Python commands.
Use the full path to the interpreter: `C:/Projects/pythonenv-lerobot/Scripts/python`

## Purpose

This repo is the autonomous learning orchestrator for the canvas world model (CWM) system. It imports from three sibling repos at `../canvas-world-model`, `../canvas-robot-control`, and `../robotic-foundation-model-tests`. See `PLAN.md` for the full implementation plan.

## Implementation starting point

Read `PLAN.md` first. It contains: context, state machine design, repo layout, module sketches, config schema, sibling-repo coupling, open design decisions, and verification plan.
